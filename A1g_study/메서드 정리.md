## 메서드 정리

### 1. 리스트

| 메서드                          | 설명                         | 시간복잡도 |
| ------------------------------- | ---------------------------- | ---------- |
| `<list>.append()`               | 마지막에 삽입                | `O(1)`     |
| `<list>.sort(reverse = False)`  | reverse = True 이면 내림차순 | `O(Nlogn)` |
| `<list>.reverse()`              |                              | `O(N)`     |
| `<list>.insert(위치인덱스, 값)` |                              | `O(N)`     |
| `<list>.count()`                |                              | `O(N)`     |
| `<list>.remove()`               |                              | `O(N)`     |



### 2. 딕셔너리

| 메서드            | 설명                                        | 시간복잡도 |
| ----------------- | ------------------------------------------- | ---------- |
| `<dict>.keys()`   | dict_keys() 객체로 반환, 리스트로 변환 필요 |            |
| `<dict>.values()` |                                             |            |
| `<dict>.items()`  |                                             |            |



### 3. 집합

| 메서드           | 설명                  | 시간복잡도 |
| ---------------- | --------------------- | ---------- |
| `<set>.add()`    | append와 동일         | `O(1)`     |
| `<set>.update()` | 원소 여러개 추가 가능 | `O(1)`     |
| `<set>.remove()` |                       | `O(1)`     |
